# Arduino---Traffic-Lamps

Az előző  "Gyalogos lámpa nyomógombbal" Arduino projektemre építettem.

EXTRA:

- Hangjelzés a gyalogos számára
- Debounce ( Állapotváltozás figyelés (HIGH -> LOW)) 
  Figyelem a gomb állapotát és ha változás történik, akkor elindul egy számláló. Ha 50ms stabil marad és valóban HIGH->LOW történik, akkor indul a folyamat.
